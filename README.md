# logo-seo-cpl
# ⚽ Cambodian Football Club Logos (SEO Website)

This project is a simple **SEO-optimized static website** designed to help football club logos appear in **Google Search and Google Images**.

---

## 🎯 Purpose

The main goal of this project is:

- 📈 Make club logos appear in **Google Images**
- 🌐 Provide **public accessible pages** for each club
- 🔍 Improve **SEO visibility** using structured HTML
- 🖼️ Serve logos from your own hosting (not Google images)

## 🚀 How It Works

- Each club has its own **static HTML page**
- Each page includes:
  - Title (SEO)
  - Description
  - Image with `alt` text
- A **sitemap.xml** helps Google index all pages
- Images are hosted locally for reliability
